Description

Parter coded with Jackie Sanchez! This experiment explores image-based evolution using p5.js. Starting from an inspiration image, the program samples average colors in random regions and draws basic shapes (rects and ellipses) to slowly recreate the image through generative mutation. I collaborated with Jackie Sanchez on the initial framework, which included capturing average color from localized regions. We then expanded the system to support more shapes and increased their number to improve coverage.

Technical

To keep color accuracy over time, the program re-samples colors every ~20 frames, avoiding performance hits while preventing drift. I also implemented an adaptive mutation rate that mimics the manual slider adjustments I was making: it starts slow, spikes briefly, then eases back—automated through a few counters and conditionals based on scoring performance.

Toward the end, I started noticing gaps in the final compositions—areas the shapes weren’t reaching. Doubling the shapes helped, tripling helped more, but a true fix would require a preprocessing pass to identify large structural regions in the source image (like Jupiter’s bands or an aurora). That’s on the to-do list for a future version.

Reflection

Overall, I’m happy with the results. The system produces visually interesting approximations, and the code is modular enough to iterate on. It’s been a fun dive into mutation-based aesthetics—and there’s plenty of room to evolve it further.
